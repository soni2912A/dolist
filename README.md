DoList — Simple To-Do App

A minimal and interactive To-Do List web application built with React, Vite and Tailwind CSS.
This app helps users add, view, mark as completed, and manage their daily tasks.

Features

Add new tasks
View all saved tasks
Save tasks in browser localStorage
Mark tasks as completed
Responsive layout for Desktop and Mobile
Built with modern React and Tailwind CSS

Technologies Used

React – UI library
Vite – Fast frontend tooling
Tailwind CSS – Utility-first CSS framework
localStorage – Client-side persistent storage
